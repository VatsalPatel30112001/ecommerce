# Ecommerce

### Installation

1. **Clone the repo:**
    ```
        git clone https://github.com/VatsalPatel30112001/ecommerce
        cd ecommerce
    ```

2. **Set up environment**
    ```
        python3 -m venv venv
        venv/bin/activate
    ```

3. **Install dependencies**
    ```
        pip install -r requirements.txt
    ```

4. **Apply migrations**
    ```
        python manage.py migrate
    ```

## Usage

1. **Start Django server**
    ```
        python manage.py runserver
    ```
